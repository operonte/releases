# Términos de uso de fasT

Al usar fasT aceptas lo siguiente:

## 1. Uso de la aplicación

fasT es una herramienta para abrir conversaciones de WhatsApp a partir de un número de teléfono. La aplicación no está afiliada a WhatsApp LLC. El uso de WhatsApp está sujeto a los términos de servicio de WhatsApp.

## 2. Formato Chile

La aplicación asume por defecto el código de país +56 (Chile). Eres responsable de introducir números correctos.

## 3. Sin garantías

La aplicación se ofrece "tal cual". El desarrollador no se hace responsable de un mal uso de la aplicación ni de los servicios de terceros (WhatsApp).

## 4. Cambios

Estos términos pueden actualizarse. Los cambios se publicarán en este documento en el repositorio releases.
