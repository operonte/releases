```
https://operonte.github.io/releases/misionapp/policies/privacy-policy.html
```

**Para activar GitHub Pages:**
1. Ir a: https://github.com/operonte/releases/settings/pages
2. Source: Deploy from a branch
3. Branch: `main` / `/ (root)`
4. Click **Save**
