# IMC App - Releases Android

## Descripción
Releases de APK y AAB para la aplicación **Calculadora IMC** en Android.

**Package:** `com.example.imcapp`

## 📦 Formatos de Release

### APK (Android Package)
- Para instalación directa en dispositivos Android
- Extensión: `.apk`
- Archivo: `imcapp-v{X.X.X}.apk`

### AAB (Android App Bundle)
- Formato recomendado para Google Play Store
- Extensión: `.aab`
- Archivo: `imcapp-v{X.X.X}.aab`
